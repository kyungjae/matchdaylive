# MatchDay Live

스포츠 사이트 초안 (sports-first IA) 프로토타입.

## 범위 (v0.1.0)
- Home IA 구조 반영: Top / Live / Scores / Fixtures / Highlights / Leagues
- LIVE strip + Scores + Fixtures + League blocks
- 모바일 반응형

## 실행
정적 파일이라 `matchdaylive/src/index.html`을 브라우저에서 열면 됩니다.
