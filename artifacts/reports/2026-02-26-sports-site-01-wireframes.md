# Sports Site Wireframes v1 (Text Mockup)

## 1) Home

### A. Top Utility Bar
- Logo
- Search
- Login/Subscribe
- Region/Language

### B. Primary Nav
- Top
- Live
- Scores
- Fixtures
- Highlights
- Leagues
- Transfers

### C. Sports Tabs (sticky)
- 축구 | 농구 | 야구 | e스포츠

### D. Hero Live Strip (Above the fold)
- LIVE NOW: 경기 3~5개 가로 스크롤
- 각 카드: 팀A vs 팀B / 스코어 / 경기시간 / 상태(LIVE, HT, FT)

### E. Main Grid (12 columns)
- Left (8):
  - Lead Story 1
  - Top Headlines 6~8개
  - Highlights (video cards)
- Right (4):
  - Today’s Scores mini-board
  - Upcoming Fixtures
  - Standings snapshot

### F. League Blocks
- Premier League / NBA / KBO / Champions League
- 블록별: Top match + 3 headlines + standings shortcut

### G. Footer
- About / Contact / Terms / Data policy

## UX Rules (Wireframe 기준)
- 3초 내 LIVE 경기 노출
- 2클릭 내 원하는 종목 진입
- 스코어/상태 정보는 텍스트 대비 최상
- 뉴스와 경기 정보의 비율 = 4:6 유지
